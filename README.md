## 📐 About the Trestle DeFi Ecosystem
Trestle is a decentralized marketplace engineered for freelancers, digital assets, and Real World Asset (RWA) integration. Built on top of Next.js, Telegram Mini-Apps, and EVM network architectures like Polygon, our economy runs on a high-velocity, three-tiered incentive pipeline:

*   **⚡ Tier 1: Vault** — The premium layer built for long-term strategic decision-making and maximum governance alignment.
*   **⛓️ Tier 2: Mine LP** — The core operational tier driving deep marketplace liquidity anchoring and yield automation.
*   **🌱 Tier 3: Stake hNOBT** — The foundational engine capturing community velocity, onboarding tasks, and early engagement rewards.

## 🛠️ Ecosystem Repositories
*   **[trestle](https://github.com/Trestle-DeFi)**: Core decentralized marketplace infrastructure optimized for web app and Telegram integrations.
*   **[trestle-website](https://github.com/Trestle-DeFi/trestle-website)**: Code framework running our primary user landing ecosystem portal.
*   **[trestle-testnet](https://github.com/Trestle-DeFi/trestle-testnet)**: Live staging environment and portal interface for testing pending updates.
*   **[trestle-reward](https://github.com/Trestle-DeFi/trestle-reward)**: Mechanics tracking distributions, active yield vectors, and active campaign logic.
